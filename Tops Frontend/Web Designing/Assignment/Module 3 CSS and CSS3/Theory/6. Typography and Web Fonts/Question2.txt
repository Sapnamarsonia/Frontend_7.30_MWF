What is the font-family property in CSS? How do you apply a custom Google Font to a webpage?

The (font-family) property in CSS is used to specify the typeface for text content. You can list multiple fonts as fallbacks, in case the first one isnâ€™t available.

body 
{
  font-family: 'Arial', 'Helvetica', sans-serif;
}


How to apply:
1. Include the Google Fonts link in the <head> of your HTML:
    <link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

2. Use the font in your CSS:
    body 
    {
        font-family: 'Roboto', sans-serif;
    }
