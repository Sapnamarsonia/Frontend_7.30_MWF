What is Web Broswer ?
A web browser is a software application used to access and view websites on the internet.
it retrieves information from the web servers and display it in a human readable format such as 
txt, images and videos.

Common example are Google Chrome, Mozilla Firefox, Microsoft EDge, safari, Opera etc

Function of web browser : 
Requests web pages using protocols like HTTP or HTTPS
Renders HTML, CSS, and JavaScript to display content
Handles user interaction like clicking links, filling forms, etc.
Manages cookies and caching to improve performance and remember user settings
